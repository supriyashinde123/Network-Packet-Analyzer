# Description

This project is a Java-based Network Packet Analyzer that uses basic Deep Packet Inspection (DPI) concepts to analyze network traffic from PCAP files. 
It extracts important packet information such as source and destination IP, ports, and protocols, and performs basic HTTP traffic detection.

# Features
- Parses PCAP files
- Extracts source and destination IP addresses
- Detects HTTP traffic
- Tracks connections between hosts

# Tech Stack
- Java
- Pcap4J
- Networking Concepts (TCP/IP, HTTP)

# How it Works
Reads PCAP → Parses packets → Extracts IP → Detects HTTP → Tracks connections

# Limitations
- Works on offline PCAP files
- Basic HTTP detection only
